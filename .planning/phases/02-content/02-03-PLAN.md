    - useEffect: reads window.matchMedia('(prefers-reduced-motion: reduce)'), sets initial state from mq.matches, adds 'change' event listener that updates state, returns cleanup removing the listener

    - 'use client' directive at top
    - Props interface FadeUpProps: children React.ReactNode, className optional string
    - Import useRef from 'react' and useReducedMotion from '@/hooks/useReducedMotion'
    - useRef<HTMLDivElement>(null) for ref
    - useEffect with [reduced] dependency array:
      - If reduced is true or ref.current is null: return early (no animation — instant appear at full opacity)
      - If reduced is false: set el.style.opacity = '0' and el.style.transform = 'translateY(16px)' INSIDE the useEffect (not in JSX render — prevents FOUC per Pitfall 2)
      - Create IntersectionObserver with threshold: 0.15
      - On entry.isIntersecting: set el.style.transition = 'opacity 400ms ease-out, transform 400ms ease-out', el.style.opacity = '1', el.style.transform = 'translateY(0)', then observer.unobserve(el)
      - observer.observe(el), return () => observer.disconnect() as cleanup
    - Only animate opacity and transform — no height, top, margin (compositor-friendly per performance.md)
    - Return <div ref={ref} className={className}>{children}</div>
  </action>
