    --- src/lib/date.ts ---

    Create the file as a pure-utility module with a single named export. No `'use client'`, no React imports, no runtime side effects.

    **REVIEW H-8 signature:** `export function formatRelativeDate(iso: string | null, now: Date = new Date()): string`

    Behavior contract:
    1. **REVIEW H-8:** if `iso === null` OR `iso === ''`, return the empty string `''` immediately and do nothing else.
    2. Parse `iso` via `new Date(iso)`. If parsing yields an Invalid Date (Number.isNaN(parsed.getTime())), return the empty string `''`.
    3. Compute the millisecond delta `now.getTime() - parsed.getTime()`. If delta < 0 (the date is in the future, which can happen for newly pushed repos with clock skew), treat it as 0 → 'today'.
    4. If delta < 24 * 60 * 60 * 1000 (24 hours), return the literal string `'today'` (UI-SPEC override).
    5. Otherwise, compute the largest unit ≥ 1 from { year, month, day } using these conversion factors:
       - 1 year ≈ 365.25 days (use a const named DAYS_PER_YEAR with the value 365.25)
       - 1 month ≈ 30.44 days (const DAYS_PER_MONTH = 30.44)
       Compute days = delta / (24*60*60*1000), then years = days / DAYS_PER_YEAR, months = days / DAYS_PER_MONTH.
       If Math.floor(years) ≥ 1: unit is 'year', value is -Math.floor(years) (negative because the event is in the past).
       Else if Math.floor(months) ≥ 1: unit is 'month', value is -Math.floor(months).
       Else: unit is 'day', value is -Math.floor(days).
    6. Construct `new Intl.RelativeTimeFormat('en', { numeric: 'auto' })` once per call.
    7. Return `formatter.format(value, unit)` — output examples: `'yesterday'`, `'3 days ago'`, `'2 months ago'`, `'last year'`, `'5 years ago'`.

    The defaulted `now` parameter exists so unit tests can pin the clock without mocking globals. Plan 03 includes a tiny Node `--test` block in the verify (see Task 3 verify) that pins `now` and asserts a few known cases — this satisfies REVIEW M-10 by replacing the bug-attractive ICU regex with deterministic unit-level checks.
